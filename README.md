# Cyberdrill Engine Room Demo

This project is a demo application for the Cyberdrill Engine Room, built with React, Vite, and Tailwind CSS.

## Features

- Modern React app with Vite for fast development
- Tailwind CSS for utility-first styling
- Modular component structure
- Example hooks and UI components
- Ready for extension and customization

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [bun](https://bun.sh/) (if using bun for package management)

### Installation

1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd cyberdrill-engine-room-demo
   ```
2. Install dependencies:
   ```sh
   bun install
   # or
   npm install
   ```

### Running the Development Server

```sh
bun run dev
# or
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).

### Building for Production

```sh
bun run build
# or
npm run build
```

### Running Tests

```sh
bun run test
# or
npm run test
```

## Project Structure

- `src/` - Main source code
  - `components/` - React components
  - `hooks/` - Custom React hooks
  - `lib/` - Utility functions
  - `pages/` - Page components
  - `test/` - Test files
- `public/` - Static assets
- `docs/` - Documentation site
